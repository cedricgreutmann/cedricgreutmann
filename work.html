3D/VR/Video Game

Historic Lens: Edinburgh VR

About


In this project I wanted to explore and create a VR experience. Something that can also teach the users something new. I created a historic map, showing how the city of Edinburgh has grown through three different time periods. This experience is based off a map found online in the National Library archives. This map was a combination of different maps, with color coded streets and buildings to show when certain areas were built.

I used this map and traced abstract 3D models off of it. to create the basic layout of the city. To add to the map I then created more detailed models of certain buildings, which the user can pick up and investigate closer.

The 3D assets were created in Maya 2016, and textured in substance painter. I finally took the models and programmed all the VR experience in the Unity game engine.

Roles


HTML, CSS coding for site using Bootstrap framework. Also coded all backend (MYSQL and PHP)

Visual Design

Concept Currency

About


Here I was creating a new design concept for a future world, that had been flooded by effects of global warming. The regulated dollar would be the world wide currency, and therefore need to mean something to all people on the planet.

These designs included notes for 50, 100, 500 and 1000. Each of the notes connected to a theme. 50 showing world piece, 100 renewable energy, 500 representing Asian world wonders, and finally 1000 representing European world wonders. The historical buildings notes were supposed to represent potentially lost monumental buildings after the flooding.

These notes were mainly created with Photoshop, using images and fractals gathered from the web, then blended together. The symbols were created in Illustrator.

Roles


Designer, created with Illustrator and Photoshop
